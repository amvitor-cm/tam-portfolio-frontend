# Tam Portfolio - Professional Frontend Developer

A modern, responsive portfolio website built with Next.js, React, and Tailwind CSS, showcasing frontend development projects and skills.

## Live Demo

**Live Site:** [https://tam-portfolio-nant.vercel.app/](https://tam-portfolio-nant.vercel.app/)

## Features

- **Modern Design**: Clean, professional interface with dark theme
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Performance Optimized**: Built with Next.js for fast loading and SEO
- **Smooth Animations**: Enhanced user experience with Framer Motion
- **Contact Form**: Integrated form with Formspree endpoint
- **Project Showcase**: Interactive project gallery with live demos

## Tech Stack

- **Framework**: Next.js 13.5.6
- **Frontend**: React 18.2.0
- **Styling**: Tailwind CSS 3.3.5
- **Animations**: Framer Motion 10.16.4
- **Deployment**: Vercel
- **Form Handling**: Formspree

## Project Structure

```
src/
├── components/
│   ├── Navbar.js      # Responsive navigation
│   ├── Hero.js        # Landing section with profile
│   ├── Skills.js      # Technologies and expertise
│   ├── Projects.js    # Project showcase gallery
│   ├── About.js       # Personal introduction
│   └── Contact.js     # Contact form and links
├── pages/
│   ├── index.js       # Main page
│   └── _app.js        # App configuration
└── styles/
    └── globals.css    # Global styles and Tailwind imports
```
